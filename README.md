# NOMBRE DEL PROYECTO
Este proyecto consiste en un mapa interactivo del mundo, donde cada país se muestra con colores personalizados y efectos visuales al pasar el cursor. La funcionalidad principal permite al usuario escribir una palabra en un cuadro de texto y, al pulsar el botón de traducir, esta palabra se traduce automáticamente al idioma oficial de cada país y se muestra dentro del mismo.   
Este proyecto combina interactividad, visualización geográfica y traducción automática, lo que lo hace útil para aplicaciones educativas, demostraciones lingüísticas o juegos didácticos de idiomas.
