Module: routes/api/game

API routes for the interactive map game mode. Includes endpoints to get random phrases/flags, validate answers, manage game statistics and query the leaderboard.

Source:
  • routes/api/game.js, line 1

(inner) GET() → {200|500}

Generates a random phrase in a random language for the game mode. The player must guess which country uses that language.

  • routes/api/game.js, line 18

  • Success - Phrase generated successfully

(inner) POST() → {200|400|500}

Saves or updates game session statistics in MongoDB. If first time, creates a new entry. If exists, updates data. When gameOver=true, calculates final score automatically.

Name Type Description
req.body Object

Request body

Properties
Name Type Attributes Description

Unique session ID (required)

Number of correct answers
attempts number <optional>

Total number of attempts
gameOver boolean <optional>

Indicates if game has ended
duration number <optional>

Game duration in seconds
* Object <optional>

Any other field from GameStats model
