Module: models/GameStats

Mongoose model to store game session statistics. Records attempts, correct answers, hints used and calculates final scores.

Source:
  • models/GameStats.js, line 1

Methods

(inner) calculateFinalScore() → {number}

Calculates the final score based on correct answers, hints used and phrases skipped. Formula: (correct * 10) - (hints * 2) - (skipped * 5)

Source:
  • models/GameStats.js, line 97
Returns:

Final score (minimum 0)

Type
number
Example
const stats = new GameStats({ correct: 10, hintsUsed: 2, phrasesSkipped: 1 });
const score = stats.calculateFinalScore();
console.log(score); // 91
