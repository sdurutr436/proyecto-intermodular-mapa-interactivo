The team has used the following technologies to build the application:

| Layer | Technology | Version |
|------|-----------|---------|
| **Frontend** | React.js | 18.2.0 |
| **Frontend Build** | Vite | 5.0+ |
| **Frontend Language** | TypeScript | 4.9.3 |
| **Interactive Map** | react-simple-maps | 3.0.0 |
| **Backend** | Node.js + Express.js | 20+ / 4.18+ |
| **Database** | MongoDB | Atlas (Cloud) |
| **ODM** | Mongoose | 7.0+ |
| **Translation APIs** | DeepL API + Google Translate | - |
| **Backend Deploy** | Railway.app | - |
| **Frontend Deploy** | Railway.app | - |
| **Monitoring** | Sentry | - |
