## Stack Tecnológico Implementado

El equipo ha utilizado las siguientes tecnologías para construir la aplicación:

| Capa | Tecnología | Versión |
|------|-----------|---------|
| **Frontend** | React.js | 18.2.0 |
| **Frontend Build** | Vite | 5.0+ |
| **Lenguaje Frontend** | TypeScript | 4.9.3 |
| **Mapa Interactivo** | react-simple-maps | 3.0.0 |
| **Backend** | Node.js + Express.js | 20+ / 4.18+ |
| **Base de Datos** | MongoDB | Atlas (Cloud) |
| **ODM** | Mongoose | 7.0+ |
| **APIs de Traducción** | DeepL API + Google Translate | - |
| **Deploy Backend** | Railway.app | - |
| **Deploy Frontend** | Railway.app | - |
| **Monitorización** | Sentry | - |
